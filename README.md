# PPT 제안서 패키지 제작

PPT 제안서 목차

1. Project Overview

프로젝트의 목표와 이벤트 기획 배경을 소개합니다.

2. Market Insight

토론토 버블티 시장 및 소비자 트렌드를 분석하고 마케팅 기회를 도출합니다.

3. Campaign Concept

파핑 보바의 특징을 활용한 이벤트 컨셉과 핵심 메시지를 설명합니다.

4. Marketing Strategy

온·오프라인을 연계한 참여형 마케팅 전략을 제시합니다.

5. Customer Journey

SNS 노출부터 매장 방문, 이벤트 참여, 재방문까지 고객 경험 흐름을 설명합니다.

6. Campaign Execution

POP TAP 게임, SNS 인증 이벤트, 리워드 프로그램 등 캠페인 실행 방안을 소개합니다.

7. Expected Results

매출 성장, 고객 증가, 브랜드 충성도 및 SNS 확산 효과를 제시합니다.

8. Benchmark & Market Data

유사 브랜드 사례와 시장 데이터를 기반으로 전략의 타당성을 설명합니다.

9. Budget Plan

캠페인 실행을 위한 예산 구성과 비용 효율성을 설명합니다.

10. Conclusion

캠페인의 핵심 가치와 기대되는 비즈니스 효과를 요약합니다.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
